2023 VASP Automation System

v.0.1.3 alpha

last update: 11th March 2023

23vaspAS - written in bash - is designed to automate various common tasks associated with the Vienna Ab initio Simulation Package (VASP) in an intuitive and user-friendly way. Regardless of whether you just want to run quick tests or start comprehensive high-throughput projects, 23vaspAS will be of great help.
